# 💁🏻 해시 테이블이란

![Hash Table](/static/images/hash-table2.png)

해시 테이블은 **`Key`와 `Value`가 쌍을 이룬 형태로 데이터가 저장되어 있는 자료구조형**을 지칭합니다. 또한, 해시 테이블은 **해시 함수를 활용해서 Key 값에 인덱스**를 배정하고, **index의 값에 데이터**를 넣는 자료 구조입니다.

배열 내 데이터도 `Key`와 `Value`로 이루어져 있기는 하나, 배열에서는 `Key`가 오직 index, 즉 숫자만 가능한 것에 비해 해시 테이블에서는 **문자열 또한 Key**가 될 수 있습니다.

---

## 🤖 해시 함수란

해시함수는 **임의의 길이의 데이터를 고정된 길이의 데이터로 매핑하는 함수**입니다. 이때 매핑 전 원래 데이터의 값을 `키 (Key)`, 매핑 후 데이터의 값을 `해시 값 (Value)`, 매핑하는 과정을 `해싱`이라고 합니다.
