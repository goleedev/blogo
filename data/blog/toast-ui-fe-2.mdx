## JS

### 새로운 이름으로 변수에 할당 할 때는 꼭 Destructuring을 사용하지 않아도 된다.

```
    <!-- Bad -->
    const changeFirstName = user.firstName;

    <!-- Good -->
    const {firstName: changeFirstName} = user;
```

## 안티 패턴

### parseInt 10진수

```
    // 10진수로 변환하는 경우라면 'Number()'를 사용하거나 '+'연산자를 붙이는 것이 더 빠름
    const month = Number("08");
    const day = +"09";
```

### 배열 길이 줄이기

```
    배열 길이를 줄이고 싶다면 length를 사용
    const numbers = ["zero", "one", "two", "three", "four", "five"];
    numbers.length = 4; // ['zero', 'one', 'two', 'three'];
```
