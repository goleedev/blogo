## 🚧 TypeScript 환경 설정

타입스크립트를 글로벌로 설치 후 특정 모듈을 사용하려면 에러가 발생한다.

```bash

Err: Cannot find module 'typescript/bin/tsc'
error Command failed with exit code 9

```

이 모듈을 계속해서 시행 하려면 타입스크립트를 글로벌 설치가 아닌 로컬로 설치해주어야 한다.

```bash

yarn global remove typescript

yarn add typescript

```
