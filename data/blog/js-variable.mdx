# 💪 변수

**특정 이름에 특정 값을 설정하는 것**을 `선언`이라고 부릅니다.

쉽게 말하면 **value 는 1이야**하고 정해주는 것입니다.

이렇게 값을 선언 할 때에 **두가지 종류**로 니늘 수 있는데, 하나는 `변수`이고 하나는 `상수`입니다.

먼저, `변수`는 **바뀔수 있는 값**을 말하며, 한번 값을 선언하고 나서 **바꿀 수 있습니다**.

한마디로 **`재할당`과 `재선언`이 가능**하다는 것이지죠.

---

# 🦾 상수

반면, `상수`는 한번 선언하고 **값이 바뀌지 않는 값**을 의미합니다.

즉, **고정적인 값을 가지는 것**이며 **`재할당`과 `재선언`이 불가능**합니다.

---

# 🤖 데이터 타입

## 숫자 (Number)

```javascript
let value = 1
```

## 문자열 (String)

```javascript
let value = 'Hello'
```

## 참/거짓 (Boolean)

```javascript
let value = true
```

## Null 과 Undefined

```javascript
let value = null
let value = undefined
```

`null`과 `undefined` 모두 **자바스크립트 상에서 값이 없음**을 나타내지만,

`null`은 **값이 없다**를 선언하는 반면 `undefined`는 **아직 값이 없음**을 의미합니다.
