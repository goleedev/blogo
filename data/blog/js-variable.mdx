## 💪 변수

특정 이름에 특정 값을 설정하는 것. 우리는 이것을 선언 이라고 부릅니다.

쉽게 말하면 이제부터 `value 는 1이야` 라고 정해주는 것입니다.

이렇게 값을 선언 할 때에 두가지 종류러 니늘 수 있는데, 하나는 변수이고 하나는 상수입니다.

만자 변수는 바뀔수 있는 값을 말합니다. 한번 값을 선언하고 나서 바꿀 수 있습니다.

한마디로 재할당과 재선언이 가능하다는 것이지죠.

---

## 🦾 상수

반면, 상수는 한번 선언하고 값이 바뀌지 않는 값을 의미합니다.

즉, 고정적인 값을 가지는 것입이며 재할당과 재선언이 불가능합니다.

---

## 🤖 데이터 타입

### 숫자 (Number)

```
let value = 1;
```

### 문자열 (String)

```
let value = 'Hello';
```

### 참/거짓 (Boolean)

```
let value = true;
```

### null 과 undefined

```
let value = null;
let value = undefined;
```

null과 undefined 모두 자바스크립트 상에서 값이 없음을 나타내지만,

null은 '값이 없다'를 선언하는 반면 undefined는 '아직 값이 없음'을 의미합니다.
