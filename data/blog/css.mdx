## 💁🏻 CSS란

CSS는 Cascading Style Sheets의 HTML의 각 요소들의 style을 정의하여 화면 등에 어떻게 렌더링하면 되는지 브라우저에게 설명하기 위한 언어입니다.

초기에는 웹의 외형을 구현하면서 HTML이 가진 정보 구조를 해치기까지 했지만, HTML5이 널리 사용되면서 HTML은 정보와 구조화, CSS는 styling의 정의라는 명확한 구분이 이루어지고

HTML이 가진 정보 구조를 해치지 않으면서 웹 사이트를 디자인할 수 있게 되었습니다.

---

## 👷🏻 CSS의 기초

CSS는 크게 선택자(selector)와 선언부(declaration block)로 이루어지며, 그리고 선언부는 다시 속성(property)과 속성 값(value)으로 이루어집니다.

선언부는 세미콜론(;)으로 속성과 속성 값을 구분하여 여러 개의 선언(declaration)을 지정할 수 있습니다.

간단히 아래와 같이 정리할 수 있습니다.

```

- 선택자

- 선언부
    - 속성: 속성 값; // 선언 1
    - 속성: 속성 값; // 선언 2
        .
        .
        .

```

위에서 설명한 것처럼 선택자에 스타일링에 대한 선언을 부여하면 됩니다.

```css
p {
  color: black;
}

.class {
  color: red;
}

#id {
  color: yellow;
}
```
