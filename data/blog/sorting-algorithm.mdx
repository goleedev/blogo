## 💁🏻 정렬 알고리즘이란

컴퓨터 과학과 수학에서 정렬 알고리즘(sorting algorithm)이란 원소들을 번호순이나 사전 순서와 같이 일정한 순서대로 열거하는 알고리즘입니다.

이는 컴퓨터에서 가장 많이 이용되는 연산 중 하나이며, 자료검색의 효율성을 측정하고 실용성과 이론 설명 등을 위해서 또한 정렬 알고리즘이 필요합니다.

이러한 정렬을 수행하는 다양한 알고리즘들이 있으며, 100개 이상의 정렬 알고리즘들이 개발되어 있습니다.

해당 글에서는 몇 가지 대표적인 정렬 알고리즘만 다룰 예정입니다.

## 🧼 버블 정렬 (Bubble Sort)

### 설명

2개의 인접한 데이터를 비교해서 앞에 있는 데이터가 뒤에 있는 데이터보다 크면 자리를 바꾸는 정렬 알고리즘입니다.

![Bubble 1](/static/images/bubble2.gif)

### 코드로 이해하기

```javascript
function bblSort(arr) {
  for (let i = 0; i < arr.length; i++) {
    for (let j = 0; j < arr.length - i - 1; j++) {
      if (arr[j] > arr[j + 1]) {
        var temp = arr[j]

        arr[j] = arr[j + 1]
        arr[j + 1] = temp
      }
    }
  }

  console.log(arr)
}
```

### 특징

- N개의 원소가 있는 경우 정렬은 N-1턴 일어납니다.

- 정렬이 1턴 일어날 때마다 가장 큰 숫자가 뒤에서 부터 1개씩 결정됩니다.

- 정렬이 1턴 일어날 때, 해당 턴 안에서 swap은 N-1-(정렬된 턴 수)번 실행되게 됩니다.

- 만약 정렬을 한 턴 실행했는데 swap이 한번도 일어나지 않은 경우, 이미 정렬된 상태이므로 로직이 일찍 종료될 수 도 있습니다.
