## 💁🏻 재귀함수란

함수가 자기 자신을 다시 호출하는 구조로 만들어진 함수입니다. 재귀함수는 종료 조건이 있어야 하며, 종료 조건을 설정해주지 않으면 무한 반복을 하게 됩니다.

재귀함수로 작성이 되는 코드는 반복문으로도 작성할 수 있습니다.

---

## 🖥️ Factorial 예시

### 반복문으로 구현

```javascript
function factorial(n) {
  var result = 1
  for (var i = n; i >= 1; i--) {
    result *= i
  }
  return result
}

factorial(n)
```

### 재귀함수로 구현

```javascript
function factorial(n) {
  if (n === 1) {
    return 1
  }
  return n * factorial(n - 1)
}

factorial(n)
```

앞서 다루어졌듯이 재귀함수의 if 구문이 없다면 해당 코드는 무한루프에 빠지게 되기 때문에,

재귀함수를 사용하는 경우 반드시 Base Case (종료 조건)이 존재해야 합니다.
