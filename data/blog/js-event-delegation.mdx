# 💁🏻 이벤트 위임이란

앞서 작성한 [`이벤트 버블링`과 `캡쳐링`](https://www.golee.tech/blog/js-event-bubbling-capturing) 을 먼저 이해해야 `이벤트 위임`을 쉽게 이해할 수 있습니다.

이벤트 위임은 실제 **바닐라 JS로 웹 앱을 구현할 때 자주 사용하게 되는 코딩 패턴**인데요. 이벤트 위임을 한 문장으로 요약해보면 **하위 요소에 각각 이벤트를 붙이지 않고 상위 요소에서 하위 요소의 이벤트들을 제어하는 방식**이라고 할 수 있습니다.

위의 내용을 코드로 작성하면 아래와 같습니다.

```html
<h1>오늘의 할 일</h1>
<ul class="itemList">
  <li>
    <input type="checkbox" id="item1" />
    <label for="item1">이벤트 버블링 학습</label>
  </li>
  <li>
    <input type="checkbox" id="item2" />
    <label for="item2">이벤트 캡쳐 학습</label>
  </li>
</ul>
```

```javascript
var itemList = document.querySelector('.itemList')

itemList.addEventListener('click', function (event) {
  alert('clicked')
})
```
