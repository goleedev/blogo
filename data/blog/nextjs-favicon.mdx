## Favicon

A favicon is a graphic image (icon) associated with a particular Web page and/or website.

Many recent user agents (such as graphical browsers and newsreaders) display them

as a visual reminder of the website identity in the address bar or in tabs.

## How to Add Favicon on NextJS

- Create `/static` Folder in Project Root

- Add `Favicon File` of Your Choice Inside

- Add Favicon Link to `Head`s

```html
<link rel="shortcut icon" href="../static/favicon.ico" />
```
