# 💁🏻 Favicond이란

![Fav](/static/images/fav.png)

_Source | `golee.tech`_

`파비콘 (favicon)` 또는 패비콘이라고 부르며 인터넷 웹 브라우저의 주소창에 표시되는 **웹사이트나 웹페이지를 대표하는 아이콘**을 의미합니다.

간단히 말하여, **플로피 디스크 이모지**가 해당 블로그의 파비콘입니다.

- favicon는 원하는 스펙 (앱도 추가할 것인지, 탭의 파비콘만 추가할 것인지 등) 떄문에 여러 가지 크기나 확장자가 필요한대요.

- 이때 무료로 [WEBSITE PLANET - Favicon Generator](https://www.websiteplanet.com/webtools/favicon-generator/)에서 쉽게 변환할 수 있습니다!


---

# ✅ Favicon 추가하기

- 프로젝트 루트에 `/static` 폴더를 추가합니다.

- 해당 폴더에 `Favicon File (e.g. favicon.ico)`를 추가합니다.

- `Head` 컴포넌트에 하단의 링크를 추가해줍니다.

```html
<link rel="shortcut icon" href="../static/favicon.ico" />
```
