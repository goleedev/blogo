# 💁🏻 DOCTYPE이란

```html
<!DOCTYPE html>
<html>
  <head> </head>
  <body></body>
</html>
```

`문서 형식 선언 (Document Type Declaration)`, 또는 **`doctype`이란 문서의 유형을 정의하기 위해 사용하는 선언문**입니다.

문서 타입 정의는 `HTML5`, `XHTML`, `HTML`의 세가지 문서 유형이 존재하며, 기술한 유형에 따라 **마크업 문서의 요소와 속성들을 처리하는 기준이 되며 유효성 검사에 이용**됩니다.

웹 문서의 시작을 알려주며 **태그보다 먼저 선언**합니다. DOCTYPE은 **웹 브라우저에서 처리할 문서가 HTML이며 어떠한 버전으로 사용했으니, 해당 방식대로 해석하라는 의미**를 가지고 있습니다.

---

# 🏁 DOCTYPE 선언의 중요성

이처럼 `DOCTYPE`을 선언하지 않으면 각각의 브라우저들이 저마다의 기준대로 랜더링을 실행해버립니다.

웹 브라우저에서는 **문서 형식 선언이 없는 HTML 문서를 쿼크 모드 (비표준 모드)로 렌더링**하지만 문서 형식 선언이 있는 HTML 문서를 **표준 모드로 랜더링**하기 때문에, 문서 형식 선언을 이용해서 **어떤 웹 페이지가 모든 웹 브라우저에서 같은 레이아웃으로 제공**되도록 할 수 있습니다.
