# 💁🏻 C++이란

`C++`은 기존의 C 언어에 **여러 가지 강력한 기능이 추가된 프로그래밍 언어**이며,

다른 언어보다 **유연성이 높으며 로우 레벨 프로그래밍이 가능**합니다.

현재 세계에서 **가장 많이 사용되는 프로그래밍 언어 중 하나**로 학습 범위가 넓고 배우기 어렵기로 유명하기도 합니다.

# 🗒️ C++ 프로그램

## 기본 구조

C++는 기본적으로 다음과 같은 구조를 가집니다.

```cpp

#include 문

#define 문

int main()

{
    명령문;
    return 문;
}

```

기본 구조에서 확인할 수 있듯이 C++ 프로그램은 **반드시 하나의 `main()` 함수**를 가져야 하는데,

이는 처음 프로그램이 실행될 때 `main()`이 있는 곳을 탐색하고 **그곳을 기점으로 실행을 시작**하기 때문입니다.

또한, 명령문은 프로그램 **동작에 대한 모든 statement**를 가르키며, 반드시 세미콜론 (;)으로 문장을 끝내야 합니다.

`return 문`은 **함수의 결과값을 반환하는 statement**로 **함수의 종료**를 의미합니다.

`#inclue문`과 `#define문`은 **preprocess 문**으로 **preprocessor에 의해 처리되는 statement**를 의미합니다.

## 기본 문법

```cpp

#include <iostream> // C++ 표준 입출력 클래스

int main (void)
{
    int num;
    std::cout << "숫자 입력 :"; // C++ 표준 출력
    std::cin >> num; // C++ 표준 입력
    std::cout << num << std::endl; // C++ 문자열 변경

    return 0; // C++ 반환문
}

```
