# 🔙 CSS 조건문 - before

지금까지 CSS에서는 해당 조건에 부합하거나 부합하지 않는 로직을 표현하지 못했습니다.

하지만 이제 `@when, @else` statement이 추가되며, **조건문을 쉽게 표현**할 수 있게 되었습니다.

---

# 🆕 CSS 조건문 - after

`@when` statement은 if statement와 동일하게, **원하는 조건을 뒤에 작성**해주면 됩니다.

또한, `@else` statement도 if-else의 **else statement**와 같이 원하는 조건을 표현해주면 됩니다.

예를 들어, 조건문으로 **device 크기에 따라 다른 스타일**을 주고 싶다면 아래와 같이 작성해주면 됩니다.

```css
@when media(max-width: 1000px) {
  // maybe mobile devices
} @else media(max-width: 700px) {
  // totally mobile
} @else {
  // tablets or desktops
}
```
