# 💁🏻 `map`, `filter`, `reduce`의 공통점

`map`, `filter`, `reduce` 메서드는 모두 ES6에서 추가된 메서드로 모두 **배열 (Array) 타입**에 사용되며, **각 요소들을 파라미터로 받아 작동**한다는 공통점이 있습니다.

---

# 🗺️ `map`

```javascript
const nums = [1, 2, 3]

nums.map((n) => n + 1)

// 출력 [2, 3, 4]
```

`map`은 배열의 **각 요소애 대해 주어진 함수를 수행한 결과를 모아 새로운 배열을 반환**하는 메서드입니다. 매개변수로 **배열의 인자를** 받으며, 해당 **인자에 변화를 준 뒤 반환**하는 식으로 사용됩니다.

`map`은 총 세개의 매개변수를 가지고 있으며, `value (현재 요소)`만 필수 매개변수이며 나머지 `index (요소의 인데스)`와 `array (호출한 원본 배열)`은 필수가 아닙니다.

---

# 🔖️ `filter`

```javascript
const nums = [1, 2, 3]

nums.filter((n) => n >= 2)

// 출력 [2, 3]
```

`filter`는 배열의 **각 요소에 대해 주어진 함수의 결과값이 참인 요소를 모아 새로운 배열을 반환**하는 메서드입니다. `filter`는 다른 메서드와는 다르게 **boolean 타입만을 반환**한다는 특징이 있으며, 리턴 값이 참인 경우에만 배열에 추가합니다.

`filter` 또한 총 세개의 매개변수를 가지고 있으며, `value (현재 요소)`만 필수 매개변수이며 나머지 `index (요소의 인덱스)`와 `array (호출한 원본 배열)`은 필수가 아닙니다.

---

# 🤲 `reduce`

```javascript
const nums = [1, 2, 3]

nums.reduce((acc, cur) => acc + cur, 4)

// 출력 10
```

`reduce`는 배열의 각 요소에 대해 **reducer 함수**를 실행하고, **하나의 결과값을 반환**합니다. 위의 코드에서 `reduce`는 값을 계속 누적하여 가지고 있는 누산기인 `acc`와 현재 요소를 가리키는 `cur`를 매개변수로 하여 **모든 요소의 합을 반환**하고 있습니다.

`reduce`는 총 네개의 매개변수를 가지고 있으며, `accumulator (리턴한 값을 저장하는 변수)`와 `currentValue (현재 변수)`만 필수 매개변수이며 `currentIndex (요소의 인덱스)`와 `array (호출한 원본배열)`은 필수가 아닙니다.
