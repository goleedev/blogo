## 💁🏻 CSS Flex란

Flex란 Flexible Box 또는 Flexbox라는 뜻으로 레이아웃을 위한 기능이며,

하나의 요소가 자신의 container가 차지하는 공간에 맞추기 위해 크기를 키우거나 줄이는 방법을 설정하는 속성입니다.

Flex 레이아웃의 기본적인 구조는 아래와 같습니다.

```html
<div class="container">
  <div class="item">box1</div>
  <div class="item">box2</div>
</div>
```

이처럼 기본 구조에서의 div.container는 flex 속성의 영향을 받는 전체 영역이자 부모 요소로 Flex Container라고 불립니다.

또한, 해당 Container의 내부 요소는 (예시의 box 1,2) 설정에 따라 각각 배치되는 자식 요소이자 Flex Item이라고 불립니다.

## ✨ Container에 적용되는 속성

### display

```css
.container {
  display: flex;
}
```

위의 코드처럼 display 속성에 flex 값을 주면, 이를 기점으로 flex layout이 적용됩니다.

### flex-direction

```css
.container {
  flex-direction: row;
  /* flex-direction: column; */
}
```

Container 내부의 요소가 배치되는 **축의 방향**을 선언하는 속성입니다.

row 값을 주면 가로로, row-reverse는 가로이지만 역순으로, column 값을 주면 세로로, column-reverse는 세로이지만 역순으로 배치됩니다.

### flex-wrap

```css
.container {
  flex-wrap: nowrap;
  /* flex-wrap: wrap; */
  /* flex-wrap: wrap-reverse; */
}
```

Container 내부의 요소가 Container width를 넘어가도록 배치되면,

해당 요소를 어떻게 줄 바꿈할지 결정하는 속성입니다.

nowrap은 기본값으 넘어가는 요소를 그대로 넘어가게 처리하며, wrap은 위에서 아래로 넘어가는 값을 줄 바꿈 해줍니다.

wrap-reverse는 아래에서 위로 넘어가도록 줄 바꿈 해줍니다.

### justify-content

```css
.container {
  justify-content: flex-start;
  /* justify-content: flex-end; */
  /* justify-content: center; */
  /* justify-content: space-between; */
  /* justify-content: space-around; */
  /* justify-content: space-evenly; */
}
```

justify-content는 가로 정렬에 대한 속성으로, flex-start는 배치의 시작점 부터 가로 정렬하며

flex-end는 배치의 종점부터 가로 정렬을 하고, center는 중간을 기점으로 가로 정렬을 합니다.

space-between은 요소들 사이에만 여유 공간을 두고 배치되며, space-around는 요소의 앞, 뒤, 그리고 요소들 사이에도 모두 여유 공간을 두고 배치됩니다

마지막으로, space-evenly 요소들 사이의 가로 간격을 동일하게 만들어 배치됩니다.

### align-items

```css
.container {
  align-items: flex-start;
  /* align-items: flex-start; */
  /* align-items: flex-end; */
  /* align-items: stretch; */
  /* align-items: center; */
  /* align-items: baseline; */
}
```

align-items는 세로 정렬에 대한 속성으로, flex-start는 배치의 시작점 부터 세로 정렬하며

flex-end는 배치의 종점부터 세로 정렬을 하고, center는 중간을 기점으로 세로 정렬을 합니다.

stretch는 기본 설정으로 요소의 높이가 컨테이너의 높이와 같게 변경된 뒤 연이어 배치됩니다.

마지막으로, baseline은 요소가 플렉스 컨테이너의 기준선(baseline)에 배치됩니다.
