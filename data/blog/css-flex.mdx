# 💁🏻 CSS Flex란

Flex란 `Flexible Box` 또는 `Flexbox`라는 뜻으로 **1차원 레이아웃을 위한 기능**이며,

하나의 요소가 자신의 **container가 차지하는 공간에 맞추기 위해 크기를 키우거나 줄이는 방법을 설정하는 속성**입니다.

Flex 레이아웃 설명을 위한 HTML 구조는 아래와 같습니다.

```html
<div class="container">
  <div class="item">box1</div>
  <div class="item">box2</div>
</div>
```

이처럼 기본 구조에서의 div.container는 flex 속성의 영향을 받는 전체 영역이자 부모 요소로 **Flex Container**라고 불립니다.

또한, 해당 Container의 내부 요소는 (예시의 box 1,2) 설정에 따라 각각 배치되는 자식 요소이자 **Flex Item**이라고 불립니다.

# 🗄️ CSS Flex Container 속성

## display

```css
.container {
  display: flex;
}
```

위의 코드처럼 `display 속성에 flex 값`을 주면, 이를 기점으로 **flex layout**이 적용됩니다.

## flex-direction

```css
.container {
  flex-direction: row;
  /* flex-direction: column; */
}
```

Container 내부의 요소가 배치되는 **축의 방향을 선언하는 속성**입니다.

`row` 값을 주면 가로로, `row-reverse`는 가로이지만 역순으로, `column` 값을 주면 세로로, `column-reverse`는 세로이지만 역순으로 배치됩니다.

## flex-wrap

```css
.container {
  flex-wrap: nowrap;
  /* flex-wrap: wrap; */
  /* flex-wrap: wrap-reverse; */
}
```

Container 내부의 요소가 Container width를 넘어가도록 배치되면,

해당 요소를 **어떻게 줄 바꿈할지 결정하는 속성**입니다.

`nowrap`은 기본값으로 넘어가는 요소를 그대로 넘어가게 처리하며, `wrap`은 위에서 아래로 넘어가는 값을 줄 바꿈 해줍니다.

`wrap-reverse`는 아래에서 위로 넘어가도록 줄 바꿈 해줍니다.

## justify-content

```css
.container {
  justify-content: flex-start;
  /* justify-content: flex-end; */
  /* justify-content: center; */
  /* justify-content: space-between; */
  /* justify-content: space-around; */
  /* justify-content: space-evenly; */
}
```

**justify-content는 가로 정렬에 대한 속성**으로, `flex-start`는 배치의 시작점 부터 가로 정렬하며

`flex-end`는 배치의 종점부터 가로 정렬을 하고, `center`는 중간을 기점으로 가로 정렬을 합니다.

`space-between`은 요소들 사이에만 여유 공간을 두고 배치되며, `space-around`는 요소의 앞, 뒤, 그리고 요소들 사이에도 모두 여유 공간을 두고 배치됩니다.

마지막으로, `space-evenly`는 요소들 사이의 가로 간격을 동일하게 만들어 배치됩니다.

## align-items

```css
.container {
  align-items: flex-start;
  /* align-items: flex-start; */
  /* align-items: flex-end; */
  /* align-items: stretch; */
  /* align-items: center; */
  /* align-items: baseline; */
}
```

**align-items는 세로 정렬에 대한 속성**으로, `flex-start`는 배치의 시작점 부터 세로 정렬하며

`flex-end`는 배치의 종점부터 세로 정렬을 하고, `center`는 중간을 기점으로 세로 정렬을 합니다.

`stretch`는 기본 설정으로 요소의 높이가 컨테이너의 높이와 같게 변경된 뒤 연이어 배치됩니다.

마지막으로, `baseline`은 요소가 플렉스 컨테이너의 기준선 (baseline)에 배치됩니다.

# 📁 CSS Flex Items 속성

Flex Items를 위한 속성들은 다음과 같습니다.

| **속성**        | **의미**                                        |
| --------------- | ----------------------------------------------- |
| **order**       | Flex Item의 순서를 설정                         |
| **flex**        | flex-grow, flex-shrink, flex-basis의 단축 속성  |
| **flex-grow**   | Flex Item의 증가 너비 비율을 설정               |
| **flex-shrink** | Flex Item의 감소 너비 비율을 설정               |
| **flex-basis**  | Flex Item의 (공간 배분 전) 기본 너비 설정       |
| **align-self**  | 교차 축(cross-axis)에서 Item의 정렬 방법을 설정 |
