# 💁🏻 고차함수란

고차함수란 **함수를 인자로 받는 함수 또는 함수를 반환하는 함수**를 뜻하며, 고차함수가 내장된 메서드에는 `map`, `filter`, `reduce`가 있습니다.

## 다른 함수를 인자로 받는 경우

```javascript
function funcCallback(num) {
  return num * 5
}

function funcGocha(func, x) {
  return func(x)
}

funcGocha(funcCallback, 5) // 25

// 함수 funcGocha는 다른 함수를 인자로 받는 고차 함수입니다.
// 함수 funcGocha의 매개변수 func는 콜백함수 funcCallback입니다.
```

## 함수를 리턴하는 경우

```javascript
function funcGocha(adder) {
  return function (num) {
    return num - adder
  }
}

funcGocha(7)(3) // -4

// funcGocha는 다른 함수를 반환하는 고차 함수입니다.
// funcGocha는 인자를 입력받아 익명 함수에 (클로저 함수) 리턴합니다.
// adder 매개변수는 익명 함수에 (클로저 함수) 이동합니다.
// 익명 함수의 매개 변수와 adder 변수의 합을 리턴합니다.
```
