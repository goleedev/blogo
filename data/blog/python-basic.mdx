# 💁🏻 파이썬이란

`파이썬 (Python)`은 1990년 암스테르담의 귀도 반 로섬(Guido Van Rossum)이 개발한 **인터프리터 언어**입니다.

파이썬의 사전적 의미처럼 **대부분의 파이썬 책 표지와 아이콘은 뱀 모양**으로 이루어져 있는데요.

개인적으로 파이썬의 아이콘이 **두 마리의 뱀**인줄 이제야 알았습니다 (머쓱타드)

---

# ✨ 파이썬 특징

## 스크립트 언어

파이썬은 컴파일 과정 없이 인터프리터가 **소스 코드를 한 줄씩 읽어 들여 곧바로 실행하는 스크립트 언어**입니다.

따라서, 컴파일 과정이 필요하지 않고 **실행 결과를 바로 확인하고 수정하면서 손쉽게 코드를 작성**할 수 있습니다.

## 동적 타이핑

파이썬은 **동적 타입 언어**입니다. 변수의 자료형을 지정하지 않고 **단순히 선언하는 것만으로도 값을 지정**할 수 있습니다.

이때, 변수의 **자료형은 코드가 실행되는 시점에 결정**됩니다.

이는 자료형 변환 시 번거로운 과정을 거치지 않아도 된다는 장점이 있지만, 코드 실행 도중 **예상하지 못한 타입으로 인한 에러가 발생할 수 있는 특징**이 있습니다.

## 플랫폼 독립적

파이썬은 `리눅스 (Linux)`, `유닉스 (Unix)`, `윈도우즈 (Windows)`, `맥 (Mac)` 등 **대부분의 운영체제 (Operating System, OS)에서 모두 동작**합니다.

운영체제별로 컴파일할 필요가 없기 때문에 **한 번 소스 코드를 작성하면 어떤 운영체제에서든 활용이 가능**합니다.

# 👍 파이썬 장점

## 인간다운 언어

인간다운 언어라고는 하지만 사실 **영어에 가까운 언어**라고 표현하는 것이 더욱 적합하다고 생각합니다.

프로그래밍이 전반적으로 **영어를 잘 하는 사람이 학습하기 (조금) 더 편한 점**이 있지만, 파이썬은 더욱 더 그 점이 부각됩니다.

파이썬은 **사람이 생각하는 방식을 그대로 표현할 수 있는 프로그래밍 언어**이기에, 개발자가 굳이 컴퓨터의 사고 체계에 맞추어서 프로그래밍을 하려고 하지 않아도 됩니다.

예를 들어, [1, 2, 3, 4]라는 배열에 1이 있을 때, 1이 존재한다고 출력하고 싶다면 다음과 같이 코드를 작성하면 됩니다.

```python

arr = [1, 2, 3, 4, 5]

for el in arr:
   print(el)

```

## 간결하고 쉬운 문법

파이썬은 **인간의 사고와 유사한 문법**을 지니고 있습니다. 그 때문에 많은 시간을 들이지 않고도 파이썬 문법을 학습할 수 있고,

프로그래밍을 전공으로 하지 않은 사람도 수일 내에 파이썬을 익혀 활용하는 것이 가능합니다.

## 빠른 개발 속도

위의 두 가지 장점을 고려했을 때, `1) 생각하는 방식을 그대로 표현할 수 있고`, `2) 간결하고 쉽게 작성할 수 있기 때문에`

파이썬을 활용하면 **비교적 빠르게 개발**할 수 있습니다.

## 높은 확장성 및 이식성

파이썬은 **다른 언어나 라이브러리에 쉽게 접근해 연동**할 수 있기 때문에 대표적인 `글루 (Glue) 언어 (혹은 접착제 언어)`로 분류됩니다.

높은 성능의 애플리케이션 개발이 필요한 경우 **C/C++과 같은 언어를 파이썬과 결합**해 사용할 수 있으며, 이런 경우에는 애플리케이션의 성능을 보장할 수 있으면서

동시에 별도 설치나 구성과정 없이 스크립트 언어의 장점 또한 함께 누릴 수 있습니다.

## 활발한 생태계

파이썬은 **수많은 표준 라이브러리를 제공**하기에 개발자는 모든 코드를 일일이 작성할 필요가 없습니다.

예를 들어, 특정 DB와 웹 서버를 연결해야 한다면 **기존에 작성된 파이썬 표준 라이브러리를 활용**하기만 하면 됩니다.

또한, 폭넓은 생태계와 활발한 커뮤니티 활동으로 인해 **빠르게 문제를 해결할 수 있고 협업할 수 있다는 강점**도 있습니다.
