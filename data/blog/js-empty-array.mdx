## Array

An array is a special variable, which can hold more than one value at a time.

## How to Empty

### 1) Create a New Empty Array

```javascript
const A = [a, b]
A = []
```

### 2) Empty Without Creating a New Array

```javascript
A.length = 0
```

## 3) Removing All Items

```javascript
while (A.length > 0) {
  A.pop()
}
```
