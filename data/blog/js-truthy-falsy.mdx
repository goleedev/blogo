# 📌 목차

![JS Truthy Falsy](/static/images/js22.png)

- **✅ `true`인 값**

- **❎ `false`인 값**

- **⚖️ 비교연산자**

---

# ✅ `true`인 값

- '0' (0을 포함하는 문자열)
- 'false' (false를 포함하는 문자열)
- `[]` (빈 배열)
- `{}` (빈 객체)
- `function () {}` (빈 함수)

---

# ❎ `false`인 값

- `undefined`
- `null`
- 0
- -0
- `NaN`
- `false`
- ''
- ""

위에 명시된 값들을 제외한 모든 값들은 `true` 입니다.

---

# ⚖️ 비교연산자

```

// all true
1 == '1';
1 == [1];
'1' == [1];

// all false (유형까지 고려하여 비교)
1 === '1';
1 === [1];
'1' === [1];

```
