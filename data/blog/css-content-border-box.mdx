# 💁🏻 Box Model이란

![CSS Box Model](/static/images/css-box.png)

CSS 박스 모델은 **HTML 요소가 웹 페이지에서 차지하는 공간을 정의한 모델**입니다.

이때, 실제 요소의 내용이 담긴 부분을 `content`, 요소를 감싸는 경계를 `border`,

border와 content 사이의 영역을 `padding`, border 바깥의 영역을 `margin`이라고 합니다.

---

# 📦 box-model 속성

box-sizing 속성은 **CSS의 테두리 영역의 크기를 결정**하는데,

box-sizing 속성 값에는 `content-box`와 `border-box`가 존재합니다.

- **content-box** : 지정한 CSS `width` 및 `height`를 컨텐츠 영역에만 적용합니다. 이때, `border`, `padding`, `margin`은 따로 계산되어 전체 영역이 설정 값보다 커질 수 있습니다.
- **border-box** : 지정한 CSS `width` 및 `height`를 전체 영역에 적용합니다. 이떄, `border`, `padding`, `margin`을 모두 합산하기 때문에 컨텐츠 영역이 설정 값보다 작아질 수 있습니다.
